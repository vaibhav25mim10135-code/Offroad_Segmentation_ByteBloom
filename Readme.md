Duality AI: Offroad Semantic Scene Segmentation

A robust semantic segmentation pipeline designed for unstructured off-road environments. This project leverages a frozen DINOv2 (Vision Transformer) backbone for rich feature extraction and a lightweight ConvNeXt-style head for efficient pixel-wise classification.
